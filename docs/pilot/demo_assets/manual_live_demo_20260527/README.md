# Manual Live Demo Assets

Ovaj folder sadrzi artefakte generisane za glavni manuelni live demo tok kroz `Catalog`, `Workspace`, `Governance` i `Benchmarks`.

## Sadrzaj

- `manual_live_demo_recording.webm`: kompletan snimak demo sekvence
- `operational_browser_e2e_summary.json`: summary fajl sa potvrdom da su svi glavni demo koraci prosli
- `screenshots/01_catalog_reuse_01.png`: approved reuse u `Catalog`
- `screenshots/02_workspace_resume_01.png`: resume draft session u `Workspace`
- `screenshots/03_catalog_diff_handoff_01.png`: diff handoff iz `Catalog` u `Workspace Review`
- `screenshots/04_catalog_stewardship_handoff_01.png`: stewardship handoff iz `Catalog` u `Governance`
- `screenshots/05_benchmarks_explanation_01.png`: benchmark comparison i explanation u `Benchmarks`
